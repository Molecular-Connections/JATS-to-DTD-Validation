The four files in this directory are non-normative supporting
documentation for the Journal Publishing Set described in version 1.0
of NISO Z39.96-2012 "JATS: Journal Article Tag Suite".

Version 1.0 of Z39.96-2012 is an ANSI/NISO Standard appproved on 
August 9, 2012 by the American National Standards Institute.


----------------------------------------------------------------------
NOTE: These files will all open directly into the folder where they 
are unzipped; they are not packed in folders. 
----------------------------------------------------------------------

jats-publishing-documentation-1.0.zip
	This file contains the HTML of the Tag Library for the version
	1.0 Journal Publishing Tag Set. A copy of this documentation is available
	online at http://jats.nlm.nih.gov/publishing/tag-library/1.0/
		
jats-publishing-dtd-1.0.zip
	This file contains the version 1.0 Journal Publishing Tag Set expressed 
	as a Document Type Definition (DTD). 
		
jats-publishing-rng-1.0.zip
	This file contains the version 1.0 Journal Publishing Tag Set expressed 
	as a RELAX NG schema (RNG; see http://relaxng.org). 
		
jats-publishing-xsd-1.0.zip
	This file contains the version 1.0 Journal Publishing Tag Set expressed 
	as a W3C XML Schema (XSD; see http://www.w3.org/XML/Schema). 
		

Details on the Journal Article Tag Suite and links to the NISO 
standard can be found at http://jats.nlm.nih.gov.
